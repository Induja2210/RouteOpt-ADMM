# RouteOpt-ADMM
Numerical Optimization for Optimal Route Selection using ADMM

## 📌 Project Overview
This project solves the shortest path problem using:
- Dijkstra's Algorithm (baseline)
- ADMM (optimization-based approach)

## ⚙️ Techniques Used
- Graph Theory
- Numerical Optimization
- ADMM (Alternating Direction Method of Multipliers)

## 📊 Results
- Dijkstra Cost: 34.72
- ADMM Cost: 34.75

- Logistics optimization
